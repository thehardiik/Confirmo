## Installation

1. Clone the repository:

```
git clone 
```

2. Navigate to the project directory:

```
cd Confirmo
```

3. Navigate to the server directory

```
cd server
```

4. Install Dependencies

```
npm install
```

5. Start the server:

```
npm run start
```

6. Opne new terminal and navigate to Trojan-Pixel:

```
cd Confirmo
```

```
cd web-app
```

8. Install Dependencies

```
npm install
```

9. Start the client:
